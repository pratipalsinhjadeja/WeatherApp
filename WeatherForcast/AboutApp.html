How to use Weather app

1.City Weather Details:

This screen will display current city weather details with five days of hourly forcast. User can see weather parameters like wind,pressure,humidity etc.
If there are no prvious searchs then you can search for new city by tapping on Select City button

2. Bookmarked Cities:

  •  On this screen user can see all previous city searches with current weather details of those cities.
  •  Weather details are like City name,current Temprature, current weather condition & last updated time.
  •  User can delete city by left swipe gesture
  •  All cities weather details can be refreshed by pulling down list.
  •  User can add new city to bookmark list by tapping on '+' button on top right navigationbar.
  •  If there are no previous searches then empty screen with Select City button will be shown. tap on it to add new city.

3. Select City:

  • Tap on the Searchbutton top right corner to Select City by text search.
  • Drag map to Select City by Map Pin.Selected City name will be shown under navigation bar.

Note : Sometimes selected city name & name on weather city will be different due to API of google & openweather map are different for same location.

4. Weather Settings:

  • User can change Weather Units from here
  • User can also remove all bookmarked cities.
